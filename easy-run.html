FORM Easy Run

Easy Run

Easy running builds the aerobic system. Nothing else should happen here.

Effort

Conversation pace. Full sentences without pausing. If breathing is elevated, slow down.

Easy is not a pace. It is a state of effort. The body moves without demand.

Form

Relax the shoulders. Let the arms drop low — wrist to hips. Feet land quietly under the hips. These mechanics are the foundation of the Ghost Protocol. Nothing is forced at this pace — if it feels like effort, it is too fast.

The Rule

Easy days stay easy. The rhythm of the week holds because the easy days hold. Nothing borrowed from recovery makes the hard days better.

Common mistake: running easy days too fast.

When It Feels Off

When the stride feels uneven or form starts to slip, slow down. Most small distortions resolve when pace drops and rhythm returns.

Finish easy. Leave the session
the same way you entered it.
